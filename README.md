<h2> Install </h2>

    - Clonar repositorio
    $ git clone https://github.com/Felipe1496/monsters-rolodex.git && cd monsters-rolodex

    - Instalar dependencias
    $ npm install

    - Rodar aplicativo
    $ npm start
